# Setup Guide - Salutations Kindred Spirit

## Overview
This project was generated by Lovable.ai and uses Vite with React 19. The project structure has been optimized to run smoothly locally.

## Installation Steps

### 1. Install Dependencies
```bash
npm install --legacy-peer-deps
```
**Note**: We use `--legacy-peer-deps` because this project uses React 19, which some dependencies don't fully support yet.

### 2. Run the Development Server

#### Option A: Run Vite Only (No Mock Server)
```bash
npm run dev
# or
npm start
```
This will start the Vite development server on http://localhost:8080

#### Option B: Run with Mock Server (Full Development)
```bash
npm run dev:full
```
This will start:
- Vite server on http://localhost:8080
- Mock API server on http://localhost:3001

#### Option C: Run Mock Server Only
```bash
npm run mock-server
```

## Troubleshooting

### Common Issues

#### 1. Port Already in Use
If port 8080 is already in use, you can change it in `vite.config.ts`:
```javascript
server: {
  port: 8080,  // Change this to any available port
}
```

#### 2. Module Resolution Errors
If you see import errors:
1. Clear node_modules and reinstall:
   ```bash
   rm -rf node_modules package-lock.json
   npm install --legacy-peer-deps
   ```

2. Check that you're in the correct directory

#### 3. Build Errors
```bash
npm run build
```

#### 4. TypeScript Errors
Since the project is mixed JS/TS, TypeScript errors can be ignored during development. The project runs fine with JS files.
